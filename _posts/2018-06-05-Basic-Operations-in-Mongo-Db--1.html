Connecting to the database:

After mongo is installed on your system you need to run the daemon or the database server and run the shell to do database operations, it is done as follows:

  1. Point to the directory where you have installed your mongo db and go to the bin folder. For e.g.: if it is installed in C: drive than go to C:\Program Files\MongoDB\Server\3.6\bin and copy the path from the address bar of the file explorer.

  2. Open the command prompt in elevated or normal mode and change the directory to the path in above step

    C:\Program Files\MongoDB\Server\3.6\bin>

  3. Run the following command to run the daemon or database server and set the location where your database files will be stored. For e.g.: if you want to store your data files in G:\MongoDB\db than give it as the command argument as follows :

    C:\Program Files\MongoDB\Server\3.6\bin>mongod -dbpath "G:\MongoDB\db"

  4. You can verify whether your database server is started or not by checking the following in the command window:

  5. After your database server is started now open another command prompt window and change the directory to the one in step 1 which points to the bin folder.

  6. Mongo provides us a shell which is used for performing operations like create database, collections, CRUD operations and many more. To start the shell run the following command in the command prompt window opened in step 5 :

    C:\Program Files\MongoDB\Server\3.6\bin>mongo

Now your database server and the shell is running, so we can proceed with some database operations.

Creating a database and collection:

The first and foremost thing to perform database operations we need to have a database , by default mongo provides a default database named test which you can check by running ‘db’ command in the mongo shell which is used to display the database in which you are currently working in . In order to create your own database run the following command:

> use firstDatabase
switched to db firstDatabase
>

‘use’ command takes an argument i.e. the name of the database which you want to create or you want to switch to which already exists. After your database is created run the following command to list your database:

> db
firstDatabase
>

The ‘db’ is an object that points to your current database in use. As we know Mongo db is a NoSQL database so there is no concept of tables, so here we have collections which are used to store data in JSON (JavaScript Object Notation) format as key value pairs. A collection is an unstructured source that holds the data in the form of documents (objects). Each object can have varying or same properties. In order to create a collection we use the createCollection method on the db object and pass it the name of the collection as an argument. Following is the command that is used to create a collection:

> db.createCollection("myFirstCollection")
{ "ok" : 1 }
>

If the collection is created successfully we receive the ok message with 1. To list the collection that you have created we use the show collections command:

> show collections
myFirstCollection

It will list all the collections in the current database in use. There might be scenario where you have created a collection by mistake and you want to delete it, In order to delete a collection drop method is used on the collection that we want to delete as follows:

> db.myFirstCollection.drop()
true
> show collections
>

As you can see that myFirstCollection is deleted. To delete the database we use the dropdatabase method on db object as follows:

> db.dropDatabase()
{ "dropped" : "FirstDatabase", "ok" : 1 }
>

It will drop the database in use.
